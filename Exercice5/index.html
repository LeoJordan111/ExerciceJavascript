Exercice 5

Je créé un tableau vide pour mettre les nouvelles valeurs.

Je fais une boucle pour parcourir l'ensemble des valeurs du tableau initial.

Je compare si la 1er valeur du tableau initial est dans le tableau filtré (sans doublon).

Pour cela, j'utilise la fonction "include".

Si la valeur n'est pas dans le tableau, je la rajoute.

Sinon je passe à la valeur suivante dans le tableau initial.

Une fois ma boucle terminé, j'affiche le tableau.
