Exercice 11

Création de 3 variables : le texte à vérifier / Un tableau avec l'alphabet / Un boolean pour le pangram.

J'utilise la méthode "split" afin de séparer chaque caractère en un tableau.

Je fais une boucle sur le tableau de l'alphabet.

Je condition la non présence d'une lettre.

S'il manque une lettre, cela vérifie la condition donc passe la variable "pangram" à false.

S'il ne manque pas de lettre, cela signifie que toute les lettres de l'aphabet sont présentes.

Le "pangram" restera à true.
